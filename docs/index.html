주요 기능

Kiko의 핵심 기능을 통해 일본어 학습을 더욱 효과적으로 할 수 있습니다.

음성 업로드 및 STT 변환

일본어 라디오 음성 파일을 업로드하면 AI가 자동으로 텍스트로 변환해 스크립트를 제공합니다. 실제 일본인의 발음과 표현을 그대로 학습할 수 있습니다.

음성 파일 업로드
OpenAI Whisper API 활용

스크립트 인터랙션

변환된 텍스트에서 모르는 단어를 클릭하면 즉시 한국어 의미와 예문을 제공합니다. 문맥 속에서 단어의 의미를 파악할 수 있습니다.

이용 방법

Kiko는 간단한 3단계로 일본어 학습을 시작할 수 있습니다.

1

음성 업로드

라디오, 팟캐스트 등 일본어 음성 파일을 업로드합니다.

2

스크립트 학습

변환된 텍스트로 모르는 단어를 클릭하며 학습합니다.

3

단어장 복습

저장한 단어들을 단어장에서 효과적으로 복습합니다.

Kiko의 장점

왜 Kiko로 일본어를 학습해야 할까요?

실제 일본어 학습

교과서가 아닌 실제 일본인들이 사용하는 자연스러운 일본어 표현을 배울 수 있습니다.

효율적인 학습

STT 기술과 단어 사전이 결합되어 원클릭으로 모르는 단어의 의미를 바로 확인할 수 있습니다.

개인화된 단어장

자신이 모르는 단어만 저장하여 맞춤형 단어장을 만들고 효과적으로 복습할 수 있습니다.

시간 절약

라디오 콘텐츠를 듣고 사전을 찾는 번거로움 없이 한 플랫폼에서 모든 학습이 가능합니다.

자주 묻는 질문

Kiko에 대해 궁금한 점이 있으신가요?

어떤 음성 파일 형식을 지원하나요?

MP3, WAV, M4A 등 대부분의 오디오 파일 형식을 지원합니다. 최대 30분 길이의 파일까지 업로드 가능합니다.

무료로 이용할 수 있나요?

기본 기능은 무료로 이용 가능합니다. 월 5개의 음성 파일까지 무료로 변환할 수 있으며, 프리미엄 요금제를 통해 더 많은 기능을 이용할 수 있습니다.

STT 변환의 정확도는 어느 정도인가요?

OpenAI Whisper API를 활용하여 약 95% 이상의 정확도를 제공합니다. 일본어 발음이 명확한 콘텐츠일수록 더 높은 정확도를 보입니다.

직접 라디오 콘텐츠를 녹음할 수도 있나요?

아직 녹음 기능은 제공하지 않습니다만, 향후 업데이트를 통해 웹사이트 내에서 직접 라디오 콘텐츠를 녹음하는 기능을 추가할 예정입니다.
